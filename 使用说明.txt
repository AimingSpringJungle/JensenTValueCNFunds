1. 安装Python或Conda，并在安装时将其加入PATH环境变量，MiniConda清华源下载地址：
	https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-py38_4.8.3-Windows-x86_64.exe
2. 换源：运行命令：
	pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
3. 完整解压此zip，记录解压路径
4. 安装依赖环境：运行命令：
	cd /d 路径		（将"路径"换为解压路径）
	pip install -r requirements.txt
5. 自定义代码：
	用记事本打开fund_crawl_capm.py
	其中的code_num表示随机选取的基金数量（默认10只）
	index_code表示指数代码（默认沪深300）
	可自行修改
6. 运行代码：
	python fund_crawl_capm.py
7. 程序将输出所爬取的无风险利率、基金代码及其与指数回归所得的alpha的t值
8. 程序会自动将原始数据输出至原始数据文件夹中
